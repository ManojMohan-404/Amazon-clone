# Amazon Clone Project

This project is a front-end clone of the Amazon homepage, built purely with **HTML** and **CSS**. It was created for educational purposes to practice core web development skills by replicating a complex, real-world user interface.

## Live Demo

A live demo of the project can be found here: [Amazon Clone Demo](https://amaz0n-clone-homepage.netlify.app/)

## üöÄ Features

The clone currently includes the following static components:

- **Header & Navigation Bar**:
  - Amazon Logo
  - "Deliver to" location selector
  - Functional search bar with a dropdown for categories
  - "Account & Lists" and "Returns & Orders" sections
  - Shopping cart
- **Secondary Navigation Panel**: A panel below the main navigation for browsing categories like "Today's Deals", "Customer Service", etc.
- **Hero Section**: The main promotional banner area.
- **Product Category Cards**: A grid layout of cards for different shopping sections (e.g., "Clothes", "Electronics").
- **Comprehensive Footer**: A multi-section footer containing links for "Get to Know Us", "Make Money with Us", and other information, mimicking the real Amazon footer.
